
Chrome :

Navigate to `chrome://version/` and get value of `Profile Path`, result is something like : `C:\Users\<Username>\AppData\Local\Google\Chrome\User Data\Default`

Then find `History` file

__________________

Edge :

Navigate to `Edge://version/` and get value of `Profile Path`, result is something like : `C:\Users\<Username>\AppData\Local\Microsoft\Edge\User Data\Default`


Then find `History` file
